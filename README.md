# Replugged Tables

A plug to render markdown tables, in discord.

## Features

- Renders Tables!!!!
  - Including column alignment
- Utility classes:
  - Table: md-table
  - Rows: md-table-row
  - Alignment: md-col-{left/center/right}
  - Heading columns: md-table-h
